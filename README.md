# Poker

##  專案簡介 
撲克牌的抽牌與下注系統!!!

##  使用方式
一鍵發牌，隨機發五張牌  
可以選取特定牌，進行重抽  
自動判斷最終牌型  
自訂押注金額進行賭注

##  創意加分
在牌型的欄位增加賭注結果  
更換背景為籌碼
